# mcp-guard

Local Firewall & Security Proxy for AI Agents.

## Objective
Focus entirely on the Local Developer Experience. Build it as a CLI-based prompt firewall that pops up interactive notifications (or a system tray icon) asking for one-click confirmation whenever an agent requests a high-risk tool call (like shell commands or writes to system files). It should require zero configuration and feel as smooth as macOS's permission dialogs.
